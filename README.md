Arduino-Logging
===============
Logging library for Arudino

Usage
-----
```
#include <Logging.h>
#define LOGLEVEL LOG_LEVEL_DEBUG
Log.Init(LOGLEVEL, 9600L);
Log.Info("Write some info %d %d", 1, 2);
```
